## 概述

双模式战略分析引擎，覆盖从任务前置分析到深度行业研究的全场景。

## 双模式

### Quick Mode — 任务前置分析
- MECE拆解
- 议题树构建
- 假设驱动验证
- 80/20优先级排序

### Deep Mode — 深度行业研究
- PEST宏观分析
- 波特五力竞争格局
- 3C分析
- 价值链拆解
- 金字塔原理报告生成

## 适用场景

- 任务前置分析
- 行业研究报告
- 竞争格局分析
- 投资决策支持
- 商业计划撰写

## 使用方式

```bash
# 在 OpenClaw 中
/skill unified-strategic-analyzer
```
