# 行业分析实战框架

> 基于《行业研究方法与案例》《行业分析方法论》提炼

## PEST分析（宏观环境）

| 维度 | 分析内容 | 数据来源 |
|:----|:---------|:---------|
| Political 政治 | 政策法规、行业监管、贸易壁垒 | 政府公报、行业白皮书 |
| Economic 经济 | GDP增速、利率、通胀、消费 | 统计局、券商研报 |
| Social 社会 | 人口结构、消费习惯、文化趋势 | 人口普查、消费报告 |
| Technological 技术 | 技术突破、研发投入、专利 | 专利数据库、学术论文 |

## 五力模型（行业竞争）

```
       新进入者威胁
           ↑
供应商议价 → 现有竞争 → 买方议价
           ↓
       替代品威胁
```

## 价值链分析

```
支持活动：公司基础 → 人力 → 技术 → 采购
基本活动：进料物流→生产→出货物流→营销→服务
```

## 3C模型（战略三角）

- **Company（公司）**：核心能力、资源、成本结构
- **Customer（客户）**：需求、行为、规模
- **Competitor（竞争）**：定位、策略、动向

## SWOT分析

```
         正面            负面
内部 │ Strengths    │ Weaknesses    │
     ├──────────────┼───────────────┤
外部 │ Opportunities│ Threats       │
```

## 标杆对比（Benchmarking）

1. 选定标杆（行业最好/直接竞品）
2. 关键维度对比（产品/价格/渠道/用户）
3. 差距分析（我差在哪）
4. 改进方向（怎么赶）

## 趋势预判矩阵

```
             确定高         确定低
影响大   │  战略优先  │  情景规划  │
         │  立即行动  │  多方案备  │
─────────┼────────────┼───────────┤
影响小   │  监控跟踪  │  暂不关注  │
         │  定期复查  │  持续观察  │
```
