# 战略分析框架

> 基于《Strategy: A View from the Top》等战略经典著作

## BCG矩阵（产品组合）

```
         高市场增长    低市场增长
高份额  │  明星（Stars）│  金牛（Cash Cow）
       │  投入增长    │  收割利润
───────┼──────────────┼─────────────────
低份额  │  问号（Question）│  瘦狗（Dog）
       │  选择性投资  │  退出/剥离
```

## 4P营销组合

- **Product（产品）**: 功能、品质、设计、品牌
- **Price（价格）**: 定价策略、折扣、付款方式
- **Place（渠道）**: 渠道选择、覆盖、物流
- **Promotion（促销）**: 广告、销售促进、公关

## STP（市场细分）

- **Segmentation（细分）**: 按地理/人口/心理/行为细分
- **Targeting（目标）**: 评估并选择目标细分市场
- **Positioning（定位）**: 在目标客户心智中建立差异
