# 麦肯锡方法 — 核心方法论

> 基于《麦肯锡方法》《麦肯锡意识》《麦肯锡结构化战略思维》提炼

## 一、MECE原则（相互独立，完全穷尽）

### 核心要求
- **相互独立（ME）**：每个分类之间不重叠
- **完全穷尽（CE）**：所有分类覆盖全部可能

### 5种拆解方法
1. **二分法**: 是/否、内部/外部、线上/线下
2. **流程法**: 按步骤：获客→转化→留存→变现
3. **要素法**: 按组成：人、财、物、信息、技术
4. **公式法**: 按公式：收入=客单价×客户数×复购率
5. **矩阵法**: 二维交叉：紧急×重要=四象限

## 二、金字塔原理

### 4原则
1. 结论先行
2. 以上统下
3. 归类分组
4. 逻辑递进

### SCQA序言结构
- **S**ituation（情境）：已知稳定状态
- **C**omplication（冲突）：打破稳定的变化
- **Q**uestion（问题）：需要回答什么
- **A**nswer（答案）：核心结论

## 三、假设驱动

### 流程
提出假设 → 设计验证 → 收集数据 → 验证/证伪 → 修正 → 深入

### 假设构建原则
- 基于已有知识，大胆假设、小心求证
- 保持多个备选假设
- 优先验证最关键假设

## 四、80/20法则

20%的原因导致80%的结果。聚焦关键少数。

应用：客户、问题、工作量、时间管理
方法：列出→量化→排序→聚焦前20%→最小处理剩余80%

## 五、麦肯锡七步成诗法

1. 定义问题（SMART）
2. 结构化问题（议题树）
3. 排定优先（80/20）
4. 制定计划
5. 分析执行
6. 综合结果
7. 沟通建议
